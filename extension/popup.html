Import Filter Settings

These filter settings can normally be left alone. They are intended to speed up GPSIO imports by filtering out irrelevant data files.

If you do not see the data you are looking for during GPSIO import, it might be in a file that was filtered out by these settings.

These settings will be used the next and every subsequent time you import from a Garmin Mass Storage Mode device using GPSIO, until the next time you change them here.
